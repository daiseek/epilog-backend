# 부하테스트용 PDF 파일 디렉토리

이 디렉토리는 EpiLog 백엔드 부하테스트에서 실제 PDF 파일을 사용하기 위한 공간입니다.

## 사용 방법

1. 테스트하고 싶은 PDF 파일을 이 디렉토리에 복사하세요
2. 부하테스트 실행 시 자동으로 이 파일들이 랜덤하게 선택되어 업로드됩니다

## 권장 파일명

- `sample_book.pdf` - 샘플 책 파일
- `test_document.pdf` - 테스트 문서
- `example_story.pdf` - 예시 스토리
- `demo_content.pdf` - 데모 콘텐츠

## 주의사항

- PDF 파일 크기가 너무 크면 업로드 시간이 오래 걸릴 수 있습니다
- 실제 책의 저작권을 확인하여 테스트 목적으로만 사용하세요
- 민감한 정보가 포함된 PDF는 사용하지 마세요

## 파일이 없을 때

이 디렉토리에 PDF 파일이 없으면 자동으로 Mock PDF가 생성되어 사용됩니다.
